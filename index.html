Forme+ Gym Local Application

This is my most recent professional developing project: A web application for Forme+ gym. It was a perfect opportunity for me to start using the new versions of web developing techniques. Designed mainly using bootstrap 4.1 with help of css3, HTML5 and jQuery for client side developement, and using PHP object oriented techniques to communicate with a MySQL database for storing Data, it will be used in the gym as soon as they are ready. It manages all of the data of the Gym: Clients, coaches (and their detailed logs using barcodes), economic statistics: Incomes, expenses, gains for a chosen period, and charts to compare periods.
